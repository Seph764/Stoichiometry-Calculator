Official Instructions to Run the Stoichiometry Calculator:

1. Run the Python 3.9.4 Installer.exe file. This will guide you to install or update Python, thereby enabling files to run via the .py extension.
*2. Run stoichiometrycalculator.py. This is the main file, and it will import from the various folders as it runs.

*If it comes up asking what you would like to open it with, right click the file > open with > Python.
